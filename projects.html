The SUTD Freshmen Orientation is a programme that allows incoming Freshmore students to get to know each other and interact ahead of their academic journey in SUTD. In 2022, the theme for the orientation programme was "Multiverse Mystery", which takes place in a dinosaur multiverse.

To reflect the orientation theme, we designed and built the website with dinosaur elements such as fossils and clawprints. To make the website more interactive, we also included a countdown timer, a responsive navigation bar, and a surprise easter egg where the information of the culprit was revealed on the last day of orientation. Figma, HTML, CSS and JavaScript were used to achieve this.

Relevant information like registration details, directions to campus, and promotional materials are included as well.

We have noticed that students tend to spend a lot of time finding a study room to use for group discussions and meetings as many of these spaces are pre-occupied. OccupEye is our solution that gives students a heads up whether a study room is vacant or occupied.

We used Figma to prototype and design the user interface of our Android app. Using Java, we built the app and linked it to a Firebase real time database with features such as user profiles, accounts, and bookmarks. The database is also tied with an OpenCV script and Yolov5 machine learning model for object and person detection.

Our project has won the audience's vote award during the ISTD Project Showcase 2023.

B3S (Brain-Bending Bomb System) is a bomb defusal game built using a field programmable gate array (FPGA) and basic electronic components. Players are required to "defuse the bomb" by finishing the two minigames before the timer runs out. Making a mistake will cost you extra time and make the bomb "explode" sooner.

Using the knowledge learnt from the Computation Structures course, we encoded our game logic using the Lucid programming language into the FPGA module. Our game utilises basic CPU architecture such as arithmetic-logic unit and control unit, as well as finite state machines to create the logical flow of the game. Input and output devices like LEDs and buttons, whose behaviours are determined by the encoded logic, are connected directly into the FPGA.

Our game was showcased at the ISTD Project Showcase 2023 and had received much positive feedback.

DUOScav is an application that aims to bring employees together by pairing up and completing mini challenges together to earn points.

Employees will be randomly assigned a partner for a certain period and several tasks to complete together. Once the tasks are completed, each pair has to submit a photo as proof of completion. Employees can also compare their progress with their peers on the leaderboard and view each others' details within the company.

Employers who act as administrators can view the employees' completed tasks and verify them to award points, as well as manage the team and task assignments for each period.

This application was developed in conjunction with DSTA BrainHack 2023 CODE_EXP hackathon, with the theme of applying gamification to the workplace. Our solution was chosen as one of the top 25 finalists in the University/Polytechnic category.
